# odin-caesar-cipher

Since there is a lot of annoying manual checking that needs to be done for a Caesar Cipher, I decided that I wanted to utilize RSpec so that it checks if my solution works for me. It actually took a while for me to figure out how to get RSpec to work but eventually I got it.

Ideally, I would be able to get it to look exactly like how they look for the odin ruby exercises (displaying which test was run and if it was passed) but I couldn't get that to work. :/ I settled with it working and checking as a good enough for now.

I found the math for calculating the new value when the shift exceeded the end a bit challenging but was able to figure it out.

Notes: Used https://cryptii.com/pipes/caesar-cipher/ to encode text to put into the tests.
